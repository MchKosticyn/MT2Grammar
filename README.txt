На вкладке "Отдельные компоненты" в разделе "Компиляторы, средства сборки и среды выполнения" выбираете "Компилятор F#"
После установки в главном окне Вы увидите "Visual Studio Build Tools 2017"
Около него нажмите "Запустить"
В консоле перейдите в папку, где лежит наш проект (у нас он называется MT2Grammar)
Проверьте, что в данном директории лежит файл "ConsoleApplication.sln"
В командной строке запустите "msbuild"
Перейдите в "bin\Debug"
Запустите "ConsoleApplication.exe"
В два текстовых файла будут выведены соответствующие грамматики для языка простых чисел: типа 0 и типа 1
