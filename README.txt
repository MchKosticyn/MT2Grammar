ЗАПУСК
Скачать и запустить:
https://www.visualstudio.com/ru/thank-you-downloading-visual-studio/?sku=BuildTools&rel=15
На вкладке "Отдельные компоненты" в разделе "Компиляторы, средства сборки и среды выполнения" выбираете "Компилятор F#"
После установки в главном окне Вы увидите "Visual Studio Build Tools 2017"
Около него нажмите "Запустить"
В консоле перейдите в папку, где лежит наш проект (у нас он называется MT2Grammar)
Проверьте, что в текущей директории лежит файл "ConsoleApplication.sln"
В командной строке запустите "msbuild"
Перейдите в "bin\Release"
Запустите "ConsoleApplication.exe"

Результатом работы программы являются 4 файла и вывод на экран:
- T0.Formation.txt (вывод простого числа в порождённой грамматике типа 0)
- T1.Formation.txt (вывод простого числа в порождённой грамматике типа 1)
- T0.Grammar.txt (порождённая грамматика типа 0)
- T1.Grammar.txt (порождённая грамматика типа 1)

Вывод на экран будет содержать два выведенных простых числа в бинарном формате.


УСТРОЙСТВО ПРОЕКТА
- Main.fs - главный файл, инициализирующий работу программы
- MT.Lp.fs - файл с МТ и LBA
- MT.ToGrammar.fs - Преобразование МТ в Т0 и порождение слова из Т0
- LBA.ToTypeOne.fs - Преобразование LBA в Т1 и порождение слова из Т1
